## Blinkit Sales Performance Dashboard

# Description:

The Blinkit Sales Performance Dashboard is an interactive Power BI dashboard designed to analyze sales performance, outlet characteristics, and product-level insights for Blinkit’s quick-commerce operations.
It provides stakeholders with a clear view of total sales, average sales, item distribution, outlet performance, and customer rating trends to support data-driven business decisions.

## Tech Stack

Power BI – Data visualization and dashboard development
SQL – Data querying and preprocessing
Microsoft Excel / CSV – Data storage and cleaning
DAX (Data Analysis Expressions) – Measures and calculated fields

# 📊 Data Source

Blinkit retail dataset (sample/analysis dataset)
Public dataset obtained from Kaggle — Blinkit Grocery Sales dataset, used for educational and analytical purposes.

Contains information about:
Item types
Outlet size and location
Sales and ratings
Item visibility

# Key Features

✅ Interactive filter panel (Outlet Location, Size, Item Type)
✅ KPI cards for quick performance overview
✅ Sales analysis by item type
✅ Fat content distribution insights
✅ Outlet size and location performance view
✅ Outlet type summary with multiple KPIs
✅ Dynamic visuals responding to slicers
